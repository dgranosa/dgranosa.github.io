GSM weather station

Description

  • https://github.com/dgranosa

  • Jan 5, 2019

To Arduino microcontroller is connected GSM SIM900 Shield used for SMS communication and DHT-22 sensor for measuring temperature and humidity.
Arduino is deployed in waterproof electric box to farm and when values get outside acceptable parameters user is notified by SMS.

Technology

  • C
  • Arduino
  • GSM
  • SIM900
  • DHT-22
