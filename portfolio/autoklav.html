Autoklav

Description

  • https://github.com/dgranosa

  • May 20, 2019

Autoklav is automation project for industrial process of sterilization and pasteurization.
It's seperated in two parts Arduino and Qt.
Arduino is microcontroller written in Arduino C that controls sensors and sends data to Qt.
Qt is frontend part of project that displays the data and gives user control of the process.

Technology

  • Microcontroller
  • C
  • Arduino
  • C++
  • Qt
  • Graph rendering
