Mobile pose estimation

Description

  • https://github.com/dgranosa

  • Jan 5, 2021

Mobile pose estimation is a mobile application that uses machine learning to get a person's position in an image.
Nowadays, when most people go to gyms and exercise there, during exercise people in their minds wander off and stop counting repetitions, which results in them not knowing where they stopped.
This paper solves this problem by counting instead of user. When the application is turned on, the camera is turned on, which the user points at himself. As long as the user exercises the application itself counts the repetitions and the user does not have to worry about it. The application remembers the maximum number of repetitions performed so far, which encourages the user to more exercises. The application works by taking a single image from the camera and passing it through a neural network that eventually gives the position of a man and thus counts the repetitions
