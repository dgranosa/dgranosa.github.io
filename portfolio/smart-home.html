Smart home

Description

  • https://github.com/dgranosa

  • May 27, 2015

Smart home is huge project containing: DIY smart switch, main hub and entretainment server.
Smart switch is done with ESP8266 as small web server on which you can control relay.
Main hub is linux server written in NodeJS with frontend for user to control multiple smart switches.
Entretainment server is running on RaspberryPi written in NodeJS and multiple Bash scripts. User is able to stream music and videos or play youtube videos.

Technology

  • C
  • Arduino
  • ESP8266
  • NodeJS
  • Express
  • Youtube-dl
