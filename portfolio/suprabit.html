This project is result of assignment for job at Suprabit. Project contains backend and frontend.
Backend is written in TypeScript with LoopBack framework and connected with PostgreSQL. On backend full CRUD REST API is implemented with authorization and authentication.
Frontend is React applcation written in JavaScript. It's done with Material UI and it enables simple communcation with backend.
Database, backend and frontend are put in separate Docker containers, and all of them are connected with docker-compose.

Technology

  • TypeScript
  • LoopBack
  • PostgreSQL
  • NodeJS
  • React
  • MUI
  • Docker
